### a MovieDemo by node

练手项目，也算是第一个比较完整的项目吧，项目基于 express + mongodb + jade

路由基本可以跑通，注释很全，几乎涵盖了各个路由的设置，作用，和所涉及的到一些模块的方法使用等

---

安装 mongodb 完成相关配置;

启动 mongodb

```js

$ npm install 

$ node app.js

```
